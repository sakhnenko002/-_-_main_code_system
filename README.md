# Аналіз процесу виявлення плагіату у текстових документах на основі методів машинного та глибинного навчання

Система автоматичного виявлення плагіату та AI-генерованого тексту, розроблена в межах бакалаврської кваліфікаційної роботи спеціальності 124 «Системний аналіз». Проєкт поєднує сучасні методи обробки природної мови (NLP), машинного та глибинного навчання для аналізу текстових документів, виявлення текстових запозичень і класифікації авторства.

## Основні можливості

* Виявлення текстових запозичень на основі семантичного пошуку
* Класифікація тексту:

  * написано людиною
  * згенеровано штучним інтелектом
* Підтримка форматів:

  * `.txt`
  * `.pdf`
  * `.docx`
  * `.md`
* Локалізація підозрілих фрагментів методом ковзного вікна
* Обчислення метрик BLEU та ROUGE-L
* Стилометричний аналіз тексту:

  * TTR (Type-Token Ratio)
  * ASL (Average Sentence Length)
* Вебінтерфейс для аналізу тексту через браузер

## Архітектура системи

Система складається з двох незалежних конвеєрів:

### 1. Виявлення AI-генерованого тексту

Використовується донавчена модель:

* RoBERTa-base
* датасет PAN 2025 Voight-Kampff

### 2. Виявлення текстових запозичень

Використовується двоетапний dense retrieval pipeline:

* SBERT embeddings
* FAISS vector search
* CrossEncoder reranking

Корпус порівняння:

* Deep Blue Machine-Paraphrased Corpus
* оригінальні та перефразовані параграфи Вікіпедії

## Основні результати

### Класифікація авторства

| Модель                 | F1-score |
| ---------------------- | -------- |
| TF-IDF + Naive Bayes   | 0.9230   |
| TF-IDF + Random Forest | 0.9290   |
| BERT-base              | 0.9564   |
| RoBERTa-base           | 0.9839   |

### Виявлення запозичень

| Метод                        | Precision | Recall |
| ---------------------------- | --------- | ------ |
| TF-IDF cosine similarity     | 0.6410    | 0.7250 |
| SBERT + FAISS                | 0.7120    | 0.9400 |
| SBERT + FAISS + CrossEncoder | 0.7747    | 0.9800 |



## Наукова новизна

* Поєднання AI authorship detection та plagiarism detection в єдиній системі
* Використання семантичного пошуку замість класичних n-грамових методів
* Адаптивний поріг виявлення для коротких текстів
* Комбінація SBERT + CrossEncoder для підвищення точності dense retrieval
