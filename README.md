# -_-_main_code_system
Система автоматичного виявлення плагіату та AI-генерованого тексту на основі методів машинного і глибинного навчання. Використовує RoBERTa для класифікації авторства та SBERT+FAISS+CrossEncoder для семантичного пошуку текстових запозичень. Підтримує аналіз txt, pdf, docx і md документів через вебінтерфейс.
